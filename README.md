# react-ts-pure-project

Create simple app with TS+React+WebPack

## Usage

Run...
```
npx react-ts-pure-project [myProjectName]
```

... where [myProjectName] is directory name witch will be created.

## Project structure

After running, you will have this
```
myProjectName
├── node_modules
│   └── ...
├── package-lock.json
├── package.json
├── src
│   ├── App.tsx
│   ├── index.tsx
│   └── public
│       └── index.html
├── tsconfig.json
├── webpack.common.js
├── webpack.dev.js
└── webpack.prod.js
```
